# ChatGPT-Mac-MenuBar
Chat with OpenAI's ChatGPT in mac menu bar like a pro.

✅ Lightweight. Only 650KB.

✅ `shift+cmd+C` to trigger popover globally

✅ **Right click** on the menu bar icon to **Clean Cookies** for login another OpenAI's account
